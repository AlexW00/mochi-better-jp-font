# Better Japanese Font

![image](https://user-images.githubusercontent.com/55558407/221356780-37435825-f99b-4372-b941-7a3b2c0329ac.jpg)

## Usage

- (install [Noto Serif JP](https://fonts.google.com/noto/specimen/Noto+Serif+JP))
- download the .css theme from [here](https://github.com/AlexW00/mochi-better-jp-font/releases/latest)
- in Mochi: settings > preferences > custom css theme > add theme
